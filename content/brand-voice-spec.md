## 語言規則

### 用廣東話，唔用書面語
- ✅ `係` / `唔係` / `咁` / `喺` / `嘅` / `佢哋`
- ❌ `是` / `不是` / `這樣` / `在` / `的` / `他們`
- ✅ `老師` `學生` `學校` `課堂` `搞掂` `唔識` `即刻`
- ❌ `教師` `學員` `機構` `課室` `完成` `不懂` `立刻`

### 句式風格
- 短句。每句唔超過 20 個字。
- 唔好用「其實」「事實上」「簡單來說」呢類廢話 filler。
- 問句收尾 > 陳述句收尾（互動性更高）
- 數字要具體：「30 秒」好過「好快」、「3 個步驟」好過「幾個步驟」

### 英文 / 中英夾雜規則
- AI 工具名稱保留英文：`ChatGPT` `Manus` `Claude` `Gemini` `prompt` `workflow`
- 平台名稱保留英文：`Threads` `IG` `Reels`
- 教育術語視情況：`workshop` / `工作坊` 兩個都可以，同一篇保持一致
- 唔好中文句子入面突然插一段英文解釋，顯得唔自然

## 四條 Content Pillar

### Pillar 1 — Save-bait 工具示範（40%）
**目標**：獲得 save / share，食 interest graph 流量，養非 follower 接觸
**格式**：Carousel（IG）+ Thread（Threads）
**核心公式**：`[具體場景] + [copy-paste prompt] + [before/after result]`
**語氣**：實用、直接、唔廢話
**例子 hook**：
- 「3 個 prompt 令 ChatGPT 即刻識教書」
- 「老師用 AI 出一份 worksheet，由 2 小時變 10 分鐘」
- 「學生唔識用 ChatGPT？呢條 prompt 幫到佢」

**禁忌**：唔好淨係講「AI 好勁」，要有實際可用嘅 prompt 或步驟

### Pillar 2 — 教育者視角（25%）
**目標**：差異化，建立 trust，工具帳號抄唔到
**格式**：Thread（Threads）+ Reels script
**核心公式**：`[真實課堂場景] + [教學洞察] + [AI 嘅角色]`
**語氣**：Terrence 親身視角，有故事，有溫度，但唔濫情
**例子 hook**：
- 「上星期我喺班房見到件事，令我改變咗對 AI 教育嘅睇法」
- 「老師驚 AI 係因為唔係怕 AI，係怕自己唔夠叻」
- 「我教過嘅學生，最驚係呢樣嘢」

**禁忌**：唔好變成純 philosophy post，尾部要有 concrete takeaway

### Pillar 3 — Founder POV / 理念（20%）
**目標**：建立品牌信任，match AE 嘅受眾，但有教學背景撐住
**格式**：Long-form thread（Threads）
**核心公式**：`[反主流觀點] + [論據] + [我哋嘅立場]`
**語氣**：有主見，唔爭議性，有建設性
**例子 hook**：
- 「AI 唔會取代老師。但唔識用 AI 嘅老師，會被識用嘅取代」
- 「教 AI 工具，係教一種新嘅識字能力」
- 「我哋教人，唔止教工具」

**禁忌**：唔好太 abstract，要接地氣

### Pillar 4 — B2B Proof（15%）
**目標**：建立學校 / 企業信任，讓 decision-maker search 到你
**格式**：單張圖 + caption（IG）+ Thread
**核心公式**：`[workshop/課程 recap] + [學員反應] + [下一個機會]`
**語氣**：低調、有質感、唔 hard sell
**例子**：
- 「上週喺 [學校/機構] 做咗個 AI 工作坊，老師嘅反應係...」
- [截圖：學員 voice message / 老師 WhatsApp 回覆]

**禁忌**：唔好 fake proof，唔好誇大數字

## Hook 公式庫

### 數字型
`[數字] 個 [動詞] 令 [persona] [結果]`
- 「3 個 prompt 令 ChatGPT 即刻識教書」
- 「5 分鐘設定，AI 幫你出下星期 5 份 worksheet」

### 問題型
`[Persona] 係唔係有呢個問題？`
- 「係老師嘅，你係咪仲係咁備課？」
- 「學生唔識用 ChatGPT？可能係 prompt 嘅問題」

### 反直覺型
`大部分人以為 [X]，但其實係 [Y]`
- 「大部分老師以為 AI 係學生用嘅，但最受益嘅其實係老師」

### 故事型（Terrence 專用）
`[具體時間地點] + [發生咗乜嘢] + [你嘅洞察]`
- 「上個月喺 Generation 教班，有個學生問咗我一條問題...」

## 禁用詞 / 禁用句式

| 禁用 | 原因 |
|------|------|
| `顛覆` `革命` `改變世界` | 太 hype，唔可信 |
| `其實好簡單` | 廢話 |
| `話咁快` `一瞬間` | 不精確 |
| `你一定要知` `必看` | 硬銷感 |
| `AI 係未來` | 太 generic |
| 全英文段落 | 唔 on-brand |
| 連續兩句 `！` | 降低可信度 |

## CTA 公式

### Comment-gate（最強互動訊號）
`留言「[關鍵詞]」，我send [deliverable] 俾你`
- 留言「教書」，我 send 全套 prompt pack
- 留言「工作坊」，我 send 學校申請 checklist

### 軟 CTA
`想知多啲？下一篇講 [具體主題]`
`試咗嘅記得話我知結果`

### B2B CTA（Profile / Story）
`想幫學校申請 AI grant？Bio 入面有 link`

## 格式模板

### Threads 單帖（Save-bait）
```
[Hook — 1 句，≤20字]

[Setup — 2-3 句背景]

Prompt 1：
「[完整 copy-paste prompt]」

結果：[before/after，要量化]

[2-3 個更多 prompt，同格式]

---
想要全套 [X] 個 prompt？
留言「[關鍵詞]」即送
```

### IG Carousel（Save-bait）
```
Slide 1: Hook（大字，唔超過 10 字）
Slide 2-5: 每 slide = 1 個 prompt + result（截圖或文字）
Slide 6: CTA（留言關鍵詞 + 下一篇預告）
```

### Thread（Problem→Solution）
```
Post 1: Hook（鎖定 persona）
Reply 1: 問題描述（2-3 句）
Reply 2: 點解會咁（洞察）
Reply 3: 解決方案（具體步驟 or prompt）
尾 Reply: Deliverable + CTA
```
